git clone -> clona um repositorio git existente

git commit -> grava as alterações no repositorio

git pull -> puxa as alterações do server e mescla com o seu

git push origin main-> empurra as alterações para o repositorio remoto

git init -> inicia um repositório git

git remote add origin URL -> para linkar o repositorio local a um remoto

git commit -m "commit inicial" -> faz o commit

git add . -> adiciona todos para o commit

rm -rf -> remove diretorio e o que rstiver dentro

git log -> mostra todos os commits anterioores
